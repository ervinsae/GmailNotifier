This is a (open source) substitute for the notifications in the standard Gmail app. It doesn't work with POP/IMAP/Exchange or other mail services.

Please read the note about donations at the bottom

Usage

  1. Use "Configure"
  2. Use "Test" for a simulation
  3. Disable notifications in Gmail to avoid duplicates

Different phones supports different vibration and LED properties. Some support none and there is nothing I can do about these issues. This app uses the documented notification APIs only. Some phones notifies only when screen is off.

Features

  • Notifications for all new mail to the inbox (unlike Gmail). Mail skipping the inbox is not supported.
  • Individual account settings
  • Configurable ringtones
  • Configurable vibration length
  • No vibration during phone calls
  • Repeating notifications
  • Status bar notifications
  • Configurable notification icon
  • LED control (if your hardware supports it)
  • Simple on/off scheduling
  • No polling, no background services
  • No auto start with system
  • Uses virtually no system resources
  • Open Source (GPL) http://android.hallerud.se

Please report any problems (except for LED issues) by mail. Market comments are useless for reporting issues.
